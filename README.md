# 📊 Retail Sales Dashboard (Power BI)

## 📌 Overview

This project presents a comprehensive analysis of retail sales data using Power BI. It focuses on identifying sales trends, top-performing products, and regional performance insights.

## 🎯 Business Problem

Retail businesses need to monitor sales performance across regions and product categories to improve profitability and optimize inventory decisions.

## 📊 Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* Sales by Region
* Category-wise Performance

## 🛠️ Tools Used

* Power BI
* CSV Dataset

## 📈 Key Insights

* West region generates the highest revenue
* Technology category contributes the most profit
* Seasonal spikes observed during Q4 sales
